Vamos criar um webapp simples, mas funcional, que atenda aos requisitos. O webapp terá as seguintes funcionalidades:

  1. Busca por etapa, data, localização, alimentação e hospedagem.

  2. Ao clicar no resultado da busca, deve abrir o aplicativo de localização do dispositivo (como Google Maps) com a direção a partir da localização atual do usuário até o local selecionado.

  3. Mostrar a distância caminhada de etapa a etapa, desde Oviedo até Santiago de Compostela, com base no documento.

  4. Separar as opções de Maiorca e Ibiza.

Estrutura do webapp:

  • Página inicial com uma barra de busca e uma lista de todas as etapas (ou todos os itens) com opções de filtro.

  • Cada item da lista terá: data, localização, detalhes de hospedagem e alimentação, e a etapa do caminho (se aplicável).

  • Ao clicar em um item, abrimos o deep link para o aplicativo de mapas (usando geo: URI scheme) com as coordenadas do local. Como não temos coordenadas no documento, teremos que geocodificar os endereços para obter as coordenadas. No entanto, para simplificar, podemos usar o endereço completo e deixar o aplicativo de mapas resolver.

Plano:

  1. Criar uma base de dados (em forma de array de objetos) com todas as informações do documento.

  2. Para cada localização, tentar obter as coordenadas (lat e long) usando uma API de geocoding (como a do OpenStreetMap Nominatim). Mas note: o webapp será estático, então teremos que pré-computar as coordenadas e armazená-las no código. No entanto, para não sobrecarregar o usuário com muitas requisições, faremos a geocoding antes e armazenaremos as coordenadas no array.

Como o documento é grande, vou extrair os dados relevantes e criar um array de objetos.

Vamos dividir em duas partes:

  • O Caminho Primitivo (14 etapas)

  • As ilhas (Maiorca e Ibiza)

Para o Caminho Primitivo, temos 14 dias de caminhada, cada dia com:

  • Data

  • Etapa (número e descrição, ex: "DIA 1 DO CAMINHO – OVIEDO → GRADO (~25 KM)")

  • Hospedagem (nome e endereço)

  • Alimentação (dicas para dieta GF)

Além disso, temos os dias de viagem até Oviedo e depois para Maiorca e Ibiza.

Como não temos as coordenadas, vamos usar o endereço completo para a geolocalização. No entanto, para melhorar, poderíamos geocodificar os endereços e armazenar as coordenadas. Mas como são muitos, e o Nominatim (geocoding gratuito) pode ter limitações, vamos fazer o seguinte:

  • Incluir um link para o mapa com o endereço completo.

Exemplo de link: https://www.google.com/maps/search/?api=1&query=endereço

Isso abrirá o Google Maps com o endereço. Alternativamente, podemos usar o schema geo: para tentar abrir diretamente no app de mapas padrão, mas nem todos os dispositivos suportam da mesma forma. Vou usar o schema geo: para dispositivos móveis, mas com fallback para o Google Maps.

Na verdade, o mais confiável é usar o Google Maps, pois o schema geo: não é suportado em todos os navegadores.

Vou optar por usar o link do Google Maps.

Então, para cada local, teremos um link como:
https://www.google.com/maps/search/?api=1&query=<endereço>

Onde <endereço> é o endereço completo, com vírgulas substituídas por %20 e espaços por %20.

Exemplo:
Endereço: "C. Covadonga n7, 33002 Oviedo, Spain"
URL: "https://www.google.com/maps/search/?api=1&query=C.%20Covadonga%20n7%2C%2033002%20Oviedo%2C%20Spain"

        function abrirNoMapa(endereco) {
            // Codificar o endereço para URL
            const enderecoCodificado = encodeURIComponent(endereco);
            
            // Verificar se é dispositivo móvel
            const isMobile = /iPhone|iPad|iPod|Android/i.test(navigator.userAgent);
            
            if (isMobile) {
                // Abrir no aplicativo de mapas nativo
                window.open(`geo:0,0?q=${enderecoCodificado}`, '_blank');
            } else {
                // Abrir no Google Maps
                window.open(`https://www.google.com/maps/search/?api=1&query=${enderecoCodificado}`, '_blank');
            }
        }

        // Função para abrir localização no mapa
        function abrirNoMapa(endereco) {
            // Codificar o endereço para URL
            const enderecoCodificado = encodeURIComponent(endereco);
            
            // Verificar se é dispositivo móvel
            const isMobile = /iPhone|iPad|iPod|Android/i.test(navigator.userAgent);
            
            if (isMobile) {
                // Abrir no aplicativo de mapas nativo
                window.open(`geo:0,0?q=${enderecoCodificado}`, '_blank');
            } else {
                // Abrir no Google Maps
                window.open(`https://www.google.com/maps/search/?api=1&query=${enderecoCodificado}`, '_blank');
            }
        }

        // Função para abrir localização no mapa
        function abrirNoMapa(endereco) {
            // Codificar o endereço para URL
            const enderecoCodificado = encodeURIComponent(endereco);
            
            // Verificar se é dispositivo móvel
            const isMobile = /iPhone|iPad|iPod|Android/i.test(navigator.userAgent);
            
            if (isMobile) {
                // Abrir no aplicativo de mapas nativo
                window.open(`geo:0,0?q=${enderecoCodificado}`, '_blank');
            } else {
                // Abrir no Google Maps
                window.open(`https://www.google.com/maps/search/?api=1&query=${enderecoCodificado}`, '_blank');
            }
        }

        function searchRoteiro(query, filter) {
            query = query.toLowerCase();
            
            return [...etapasCaminho, ...voos, ...maiorca, ...ibiza].filter(item => {
                const matchesFilter = filter === 'all' || 
                                     (filter === 'etapa' && item.etapa > 0) ||
                                     (filter === 'hospedagem' && item.tipo === 'hospedagem') ||
                                     (filter === 'alimentacao' && item.alimentacao && item.alimentacao.includes('GF')) ||
                                     (filter === 'voo' && item.tipo === 'voo') ||
                                     (filter === 'carro' && item.tipo === 'car-rental') ||
                                     (filter === 'maiorca' && (item.tipo === 'maiorca' || item.tipo === 'car-rental' || item.tipo === 'hospedagem' && maiorca.includes(item))) ||
                                     (filter === 'ibiza' && (item.tipo === 'ibiza' || item.tipo === 'hospedagem' && ibiza.includes(item)));
                
                const matchesQuery = !query || 
                    item.titulo.toLowerCase().includes(query) ||
                    item.data.toLowerCase().includes(query) ||
                    item.localizacao.toLowerCase().includes(query) ||
                    (item.hospedagem && item.hospedagem.toLowerCase().includes(query)) ||
                    (item.alimentacao && item.alimentacao.toLowerCase().includes(query)) ||
                    (item.descricao && item.descricao.toLowerCase().includes(query)) ||
                    (item.tipo === 'voo' && (
                        item.companhia.toLowerCase().includes(query) ||
                        item.numeroVoo.toLowerCase().includes(query) ||
                        item.partida.toLowerCase().includes(query) ||
                        item.chegada.toLowerCase().includes(query)
                    )) ||
                    (item.tipo === 'car-rental' && (
                        item.detalhes.veiculo.toLowerCase().includes(query) ||
                        item.detalhes.locadora.toLowerCase().includes(query) ||
                        item.detalhes.voucher.toLowerCase().includes(query)
                    ));
                
                return matchesFilter && matchesQuery;
            });
        }

Opção 1: GitHub Pages (Recomendada)

  1. Crie uma conta no GitHub (se ainda não tiver uma)

    • Acesse github.com e clique em "Sign up"

  2. Crie um novo repositório

    • Clique no ícone "+" no canto superior direito

    • Selecione "New repository"

    • Dê um nome ao repositório (ex: "caminho-primitivo")

    • Marque a opção "Public"

    • Clique em "Create repository"

  3. Faça upload do arquivo

    • No seu repositório, clique em "Add file" > "Upload files"

    • Arraste o arquivo HTML ou clique para selecioná-lo

    • Adicione um arquivo README.md se desejar

    • Clique em "Commit changes"

  4. Habilite o GitHub Pages

    • Vá para a aba "Settings" do seu repositório

    • No menu lateral, clique em "Pages"

    • Em "Source", selecione a branch "main" e a pasta "/root"

    • Clique em "Save"

  5. Acesse seu site

    • Após alguns minutos, seu site estará disponível em:

    • https://[seu-usuario].github.io/[nome-do-repositorio]

Opção 2: Netlify (Alternativa Fácil)

  1. Crie uma conta no Netlify

    • Acesse netlify.com

    • Clique em "Sign up" e crie uma conta

  2. Faça o deploy do site

    • Na dashboard do Netlify, clique em "Add new site" > "Deploy manually"

    • Arraste e solte o arquivo HTML ou faça o upload

  3. Acesse seu site

    • O Netlify fornecerá automaticamente uma URL para seu site

            } else {
                // Abrir no Google Maps
